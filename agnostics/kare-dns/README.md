# Kare DNS for K8TRE

This directory contains the Kubernetes manifests for deploying Kare DNS within the K8TRE environment. Kare DNS is a custom DNS solution for managing DNS records in the K8TRE platform.

## Structure

- `k3s/` - K3S-specific configurations

## Vendor Differences

Currently, Kare DNS is only configured for K3S deployments in the K8TRE platform.
